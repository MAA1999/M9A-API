程序化访问

📋 Manifest 文件列表

api/resource/data/manifest.json

包含所有文件的元数据（文件名、路径、大小、更新时间）
程序可以读取此文件自动发现并下载所有数据文件

💡 示例：使用 Python 批量下载所有文件 
import requests
import json

# 读取 manifest
base_url = 'https://maa1999.github.io/M9A-Act-Data'
manifest = requests.get(f'{base_url}/api/resource/data/manifest.json').json()

# 下载所有文件
for file in manifest['files']:
    url = f"{base_url}/api/resource/data/{file['path']}"
    response = requests.get(url)
    with open(file['name'], 'wb') as f:
        f.write(response.content)
    print(f"Downloaded: {file['name']} ({file['size']} bytes)")

print(f"Total: {len(manifest['files'])} files")
print(f"Last updated: {manifest['updated']}")
